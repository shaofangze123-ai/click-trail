# Click Trail — 点击轨迹

Chrome / Edge 浏览器扩展，在网页上留下点击轨迹高亮标记，帮你记住编辑到哪一行。

专为 OA 系统、后台管理等大量相似行编辑场景设计。

## 解决什么问题

在 OA 系统中编辑流程权限、审批配置等页面时，经常需要逐行点击编辑。行与行内容高度相似，编辑几行后就忘了刚才编辑到哪里。

Click Trail 会在你每次点击的位置留下一个高亮标记，最新的标记最亮，旧标记逐渐变淡，帮你一眼看清编辑轨迹。

## 功能特性

- **点击即标记** — 点击页面任意元素，自动在该行/区域留下高亮覆盖层
- **渐变轨迹** — 保留最近 N 次点击标记（默认 5 次），越新越亮，越旧越淡
- **序号标注** — 每个标记带有序号，清晰显示点击顺序
- **跨页面保留** — 离开页面再回来，标记自动恢复（7 天内有效）
- **智能定位** — 自动识别表格行、列表项等行级元素
- **SPA 支持** — 监听路由变化，单页应用也能正常工作
- **iframe 支持** — 在 iframe 内的页面同样生效
- **Popup 控制** — 一键开关、调节保留数量、清除当前页标记

## 安装方式

### 本地加载（开发者模式）

1. 下载或克隆本项目
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角的 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择本项目文件夹

Edge 浏览器同理，访问 `edge://extensions/`。

## 使用方法

1. 安装后，扩展会自动在所有网页上生效
2. 点击页面上的任意元素，该位置会出现蓝色高亮标记
3. 继续点击其他位置，旧标记变淡，新标记最亮
4. 点击浏览器工具栏的扩展图标，可以：
   - 开关追踪功能
   - 调整保留标记数量（1-20）
   - 清除当前页面的所有标记
